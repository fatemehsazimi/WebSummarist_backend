# Web Summarizer Backend

A fast and simple **backend API** for summarizing web pages, supporting both standard websites and JavaScript-heavy websites.  
Built with **FastAPI** and **HuggingFace Transformers** models.

---

## Features
- Web scraping using **Playwright** for JavaScript-enabled sites.
- Fallback scraping with **Trafilatura** for simple sites.
- Text summarization using **BART / HuggingFace** models.
- Error handling and returning clear error messages.
- Modular and clean structure, ready for further development.
